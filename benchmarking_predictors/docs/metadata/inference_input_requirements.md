# Inference input files and required columns

This is a consolidated checklist of **user-provided input files** needed to run the inference scripts under `scripts/inference/`.

## 1) Core metadata table (all SNV/deletion inference except SpliceTransformer parquet-only)

**File name**: `opensplice_predictors_benchmarking_variant_metadata.tsv`

**Required columns (SNV inference):**
- `ensembl_exon_id`
- `variant_id`
- `nt_seq`
- `exon_length`

Used by:
- `spliceai_genome_snvs_inference.py`
- `spliceai_minigene_snvs_inference.py`
- `pangolin_genome_snvs_inference.py`
- `pangolin_minigene_snvs_inference.py`
- `alphagenome_minigene_mode_snvs_inference.py`

**Required columns (deletion inference):**
- `ensembl_exon_id`
- `variant_id`
- `nt_seq`
- `exon_length`
- `start`
- `length`
- `end`

Used by:
- `spliceai_genome_deletions_inference.py`
- `spliceai_minigene_deletions_inference.py`
- `pangolin_genome_deletions_inference.py`
- `pangolin_minigene_deletions_inference.py`
- `alphagenome_minigene_mode_deletions_inference.py`

## 2) Exon context table (genome-mode models)

**File name**: `opensplice_predictors_benchmarking_exon_metadata.tsv`

**Required columns (SpliceAI/Pangolin genome SNV+deletions):**
- `ensembl_exon_id`
- `up_5k`
- `wt_seq`
- `down_5k`

Used by:
- `spliceai_genome_snvs_inference.py`
- `spliceai_genome_deletions_inference.py`
- `pangolin_genome_snvs_inference.py`
- `pangolin_genome_deletions_inference.py`

**Required columns (AlphaGenome genome mode):**
- `ensembl_exon_id`
- `strand`
- `start_exon`
- `end_exon`

Used by:
- `alphagenome_genome_mode_snvs_inference.py`
- `alphagenome_genome_mode_deletions_inference.py`

## 3) SpliceTransformer inference parquet

**File name**: `splice_transformer_genomic_variant_input.parquet`

**Required columns (genomic-mode script):**
- `ensembl_exon_id`
- `ID`
- `spliceai_seq_genome`
- `nt_seq`

Used by:
- `splicetransformer_genomic_inference.py`

**Required columns (minigene-mode script):**
- `ensembl_exon_id`
- `ID`
- `spliceai_seq_minigene`
- `nt_seq`

Used by:
- `splicetransformer_minigene_inference.py`

## 4) AlphaGenome genome-mode per-exon VCFs

**File form**: one VCF per exon in a directory (e.g. `*.vcf`)

**Required VCF columns:**
- `CHROM`
- `POS`
- `ID`
- `REF`
- `ALT`

Used by:
- `alphagenome_genome_mode_snvs_inference.py`
- `alphagenome_genome_mode_deletions_inference.py`

---

## Note on outputs

Inference output file names/locations are created by each script configuration and should be generated by the user at runtime. This checklist is only for validating required **input** files/columns before inference.
